## 📖 About

This is my personal **developer portfolio** — a single-page, tab-based site built with plain **HTML, CSS, and JavaScript**. It's a dark-themed, sidebar-driven layout that brings together everything about me as a Computer Science & Business Systems student: my background, skills, resume, live projects, and achievements — all in one clean, responsive page.

I'm a CSE (Business Systems) student at **PSG Institute of Technology and Applied Research**, building technology that solves real problems at scale — from a campus printing platform used by 4,000+ students to ML-powered cricket decision systems.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🧭 **Tab-Based Navigation** | About, Resume, Projects, and Awards sections switch instantly without page reloads |
| 📌 **Sticky Sidebar** | Persistent profile card with photo, contact info, education snapshot, and social links |
| 🎨 **Custom Dark Theme** | Lime/cyan accent palette (`#c8ff00`, `#00e5ff`) on a near-black background for a modern, techy feel |
| 📱 **Responsive Layout** | Sidebar collapses gracefully on smaller screens |
| 🖋️ **Custom Typography** | Google Fonts — *Syne* for headings, *DM Sans* for body text |
| ⚡ **Zero Dependencies** | Pure HTML/CSS/JS — no frameworks, no build step, loads instantly |

## ⚙️ Running Locally

No build tools or dependencies needed — it's a static single-file site.

```bash
git clone https://github.com/Soorya-SS-01/Portfolio.git
cd Portfolio
```

Then simply open `index.html` in your browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Visit `http://localhost:8000` to view it.
